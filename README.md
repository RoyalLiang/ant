### 关于

---

这是一款Windows GUI小工具，用于批量重命名指定文件夹下的所有内容。功能包括：
1. 自定义追加前后缀
2. 关键词列表指定删除，支持文件夹
3. 关键词列表指定替换，支持文件夹

目前只是一个初始化版本，后续根据需要进行优化。
